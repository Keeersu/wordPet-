Modai API 概览

API Key 鉴权

C 端 API 使用 API Key 进行鉴权，适用于客户端应用集成。

鉴权方式：

Authorization: Bearer modai_xxxxx

或查询参数：

?api_key=modai_xxxxx
去申请 API Key

MCP 接入（Beta）

平台提供 MCP Server 适配层（服务端部署），支持将本文档中的/api/web/ai/*能力映射为标准工具调用。客户端通过 HTTP 连接，无需安装源码。

服务端点：

POST https://conan-modai-studio.zhenguanyu.com/conan-modai-studio/api/mcp

Cursor 配置示例（mcp.json）：

{
  "mcpServers": {
    "modai": {
      "url": "https://conan-modai-studio.zhenguanyu.com/conan-modai-studio/api/mcp",
      "headers": {
        "Authorization": "Bearer modai_xxxxx"
      }
    }
  }
}

工具命名规范：

image_generate_doubao
image_edit_gemini
video_generate_sora
video_animate_sora
video_get_status_sora
tts_generate_azure
tts_list_voices_minimax
music_generate_minimax
music_status_minimax
mcp_metrics

鉴权方式：API Key 在连接配置的 Authorization Header 中提供，工具调用时自动透传。

健康检查：GET /api/mcp 返回服务状态和工具数量。
